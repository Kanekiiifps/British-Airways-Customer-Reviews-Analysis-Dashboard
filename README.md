Project Overview
This is an interactive Tableau dashboard that analyzes customer reviews for British Airways. It helps visualize key insights like overall ratings, trends over time, ratings by country, and performance by aircraft type. The data comes from public sources like Skytrax or Kaggle datasets on airline reviews.
Why This Project?

Goal: To showcase data visualization skills using Tableau for analyzing customer feedback in the airline industry.
Key Insights from the Dashboard:
Overall average rating: Around 4.2 out of 5.
Ratings trend: Shows fluctuations over months/years (e.g., dips in certain periods).
Top-rated aircraft: Boeing 747-400 at 4.7.
Geographic breakdown: Ratings vary by country, with a world map view.
Filters: Interactive options for metrics (e.g., overall rating, cabin staff), seat type, traveler type, and more.

This project demonstrates skills in data cleaning, visualization, and storytelling with tools like Tableau.

Data Source

Dataset: British Airways customer reviews (e.g., from Kaggle: British Airways Reviews or similar public datasets).
Time period: Covers reviews from March 2016 to October 2023 (based on the dashboard).
Features: Includes ratings for entertainment, food, ground service, seat comfort, value for money, and more.
